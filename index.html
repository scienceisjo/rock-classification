🪨 암석 분류 알고리즘 — 사용 방법

📌 학습 목표

암석을 화성암·퇴적암·변성암으로 분류하는 기준을 분류 알고리즘을 직접 완성하며 이해합니다.

⭐ 쉬움 vs 🔥 어려움

  • 쉬움: 분류기준 카드가 완전한 텍스트로 제공됩니다. 카드 뱅크에서 골라 빈 칸에 배치하세요.
  • 어려움: 분류기준 카드의 핵심 단어가 ___ 빈칸으로 가려집니다. 어떤 분류기준인지 스스로 판단하여 올바른 칸에 배치하세요. 카드 순서도 매번 섞입니다.

🖱️ 사용 방법

  • 왼쪽 카드 뱅크에서 카드를 드래그하여 점선 빈 칸에 놓으세요.
  • 같은 카드를 여러 곳에 중복 사용할 수 있습니다.
  • 놓은 카드를 되돌리려면 ✕ 버튼을 누르세요.
  • 정답 확인으로 초록(정답) / 빨강(오답) 확인.
